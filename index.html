Current Weather Observation Charts

These are short summary charts of the current weather observations for the selected location and API.

The availability of data for any given location or api is random. Selected charts will hide themselves if no data is available.

See the Readme and Ghcnd documentation as well as console.log for additional information on the data, data codes, and available chart types for a given location.

Short Weather History Charts

These short summary charts of a subset of available historical data for the current location and API.

The availability of data for any given location or api is random. Selected charts will hide themselves if no data is available.

See the Readme and Ghcnd documentation as well as console.log for additional information on the data, data codes, and available chart types for a given location.

Climate History Charts

These charts are scrollable and include all available historical data for the current location and API. The first series of charts are temperature charts using various averaging techniques. For a good example of global warming it's recommended to set the location to "Boston" and view the highlighted chart with avg(min/max) temperatures (TMXN) averaged by months. The latter charts are snowfall (SNOW) and snow depth (SNWD).

The availability of data for any given location or api is random. Selected charts will hide themselves if no data is available.

See the Readme and Ghcnd documentation as well as console.log for additional information on the data, data codes, and available chart types for a given location.
